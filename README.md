dti
===

The goal of dti is to provide tools to analysis of Diffusion Weighted Imaging (DWI) Data.

Installation
------------

You can install `dti` from GitHub with:

``` r
# install.packages("devtools")
devtools::install_github("muschellij2/dti")
```
